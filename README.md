### Setup environment & Run tests

* requirements
    * Go version must be at least 1.12 (Modules)

1. Testing in local

```
$ make test
```

2. Testing with [Circle CI CLI](https://circleci.com/docs/2.0/local-jobs/)

```
$ circleci build --job test
```
